# dumi-site-template

A static site base on [dumi](https://d.umijs.org).

## Development

```bash
# install dependencies
$ yarn install

# start dev server
$ yarn start

# build docs
$ yarn run build
```

## Deploy

Visit https://vercel.com/new to deploy your site.
